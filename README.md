# 轻盈生活

一个纯静态、可添加到 iPhone 主屏幕的公开 PWA，把四个原有小工具重新编排成一张“今日执行台”，不是简单并排放置四个入口：

- 今日执行：把餐单三餐和三日循环放在同一张卡片中，分别记录“餐单打卡”和“循环打卡”。
- 首页食物称重：输入食物名称后按关键词自动判断固体、液体、酸奶/粥、果蔬或坚果，也可以手动修正。
- 首页晚饭秤：直接输入早体重、晚间秤重和衣服重量，查看净差值与晚饭建议。
- 详细记录：需要时再进入完整餐单趋势、称重历史、晚饭历史和循环规则页。

## 数据与隐私

应用不需要账号或后端。用户输入只写入当前浏览器的 `localStorage`，不会上传到 GitHub 或其他服务器。首页可一次导出或恢复四个模块的数据。

四个旧站点与合并版都在 `zjie1979.github.io` 同一网站域名下，并继续使用原来的存储键，因此同一设备、同一浏览器中的旧记录会自动继续使用。换手机、换浏览器或清理网站数据时不会自动迁移，请先用首页的统一备份功能导出数据。

## iPhone 安装

用 Safari 打开正式地址，点击“分享”，选择“添加到主屏幕”。
