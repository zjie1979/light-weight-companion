# 轻盈生活

一个纯静态、可添加到 iPhone 主屏幕的公开 PWA，把四个原有小工具收进同一个入口：

- 餐单打卡：三餐、体重、趋势与两套饮食计划。
- 饮食称重：分类折算、每日上限、带壳食物补录与历史。
- 晚饭秤：早晚体重、衣服重量、净差值与晚餐建议。
- 三日轻盈循环：16+8、每天两顿主餐、可选早餐与轮次进度。

## 数据与隐私

应用不需要账号或后端。用户输入只写入当前浏览器的 localStorage，不会上传到 GitHub 或其他服务器。首页可一次导出或恢复四个模块的数据。

旧站点与合并版属于不同网站地址，浏览器不会自动迁移旧站点中的本地数据。

## iPhone 安装

用 Safari 打开正式地址，点击“分享”，选择“添加到主屏幕”。
